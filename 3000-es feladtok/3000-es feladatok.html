A hírhedt 3000-es feladatok

Szerintem ebben az évben a legtöbbet a 3000-es feladatokkal foglalkoztam. Az év alatt nagyon sok házit kaptunk belőlük. Itt a lényeg az az volt, hogy egy lista számból kellett adatokat megmondani (pl.:Mi a legnagyobb szám, vagy a legkisebb). Elősször csk 11 feladatot kaptunk majd lassan egyre többet, amíg eljutottunk 90 megoldandó feladatig (3000N, az volt a csúcs), viszont, szrencsére itt már egy fügvénnyel több kérdésre is tudtunk válaszolni.

Itt vannak az elemi programozási tételek, amiket használtunk az órán és a házi feladatokban is:

Ezek segítségével oldottuk meg a 3000-es feladatokat

Hány eleme van a listának? Keressük egy lista összes elemének összegét! Keressük egy lista legnagyobb elemét! Készítsd el az a megadott lista alapján azt a listát, amelyben csak a pozitív elemek szerepelnek! Van egy negatív szám a sorozatban. Melyik a első ilyen szám? Add meg a lista első negatív számának helyét! Amennyiben nincs ilyen, a válasz legyen -1! Csoportosítsd a listában szereplő számokat az utolsó számjegyük szerint!

Itt egy kód:

 static int Hany_paros(List lista)
        {
            int db = 0;
            foreach (int elem in lista)
            {
                if (elem % 2 == 0)
                {
                    db++;
                }
            }
            return db;
        }
